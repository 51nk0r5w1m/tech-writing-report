# Tech Writing Report

Source files for the final technical writing report on PageIndex and long-document retrieval.

## Files

- `report.typ`: main Typst source
- `report.md`: Markdown draft/reference version
- `assets/`: SVG assets used by the report

## Build

```bash
typst compile report.typ report.pdf
```

## Notes

The repository tracks the source project. The locally rendered PDF can be regenerated from `report.typ`.
